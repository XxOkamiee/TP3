Explication brève du projet

Nous avons décidé de faire un site web en un type de guide touristique pour informer les futurs touristes qui comptent faire un voyage en France notamment dans la ville de Paris sur ses nombreux lieux touristiques très populaire. S’ils désirent avoir plus d’informations, ils pourront se fier sur ce guide.
Nos deux sites web qui sont nos concurrents sont les suivants : Office de Tourisme de Paris et TripAdvisor. Nous nous sommes inspirés de ces deux sites web pour la création de notre site web dans son aspect visuel et pour le contenu qu’elle englobe.

Aspect visuel

Dans toutes les pages il y a un contraste entre les couleurs bleu, blanc et rouge. La raison de ces contrastes était pour représenter le drapeau français. Blanc pour la monarchie et bleu, rouge pour les couleurs de la ville de Paris. En passant dans la première catégorie, la police était importante. La raison pour l’utilisation de cette police était pour son aspect visuel qui donne du confort au lecteur quand il lit le texte. Dans la deuxième catégorie, le groupement du texte est très importante car, il y en a beaucoup et nous ne voulons pas que le lecteur se perd dans sa lecture. Dans la troisième catégorie, les icones étaient importantes à placer pour que le lecteur puisse se retrouver dans la page sur certaine informations spécifiques.

Contenu du site

Pour la première catégorie, nous allons parler de l’homogénéité. Pour qu’un site web soit considéré « homogène », il faut qu’il se trouve une forme de constance sur toutes les pages du site. Sur notre site, cette constance sera trouvée dans le fait que sur chaque page nous avons des images représentants Paris et que le thème des couleurs reste le même sur chaque page. Pour la deuxième catégorie, ce sera sur la concision de notre site web. La concision dans le web est le fait d’écrire un très petit nombre de mot pour expliquer beaucoup. Nos pages ne contiennent pas trop de texte, donc le client/lecteur ne devrait pas avoir de difficulté à comprendre ce que le site web offre comme service. Pour la troisième catégorie, ce sera sur le guidage de notre site web. Le guidage se sert comme moyen à guider l’utilisateur. Dans notre site web, le lecteur se trouve avec 4 liens directs au-dessus de chaque page pour se diriger aux autres pages.
